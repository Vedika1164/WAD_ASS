PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics I: Calculus, Matrices, and Differential Equations
  • Engineering Physics: Basics of mechanics, optics, and quantum physics
  • Engineering Chemistry: Chemical bonding, corrosion, and water treatment
  • Basic Electronics: Semiconductors, diodes, transistors
  • Programming: Introduction to C Programming

Second Year (SE)

  • Data Structures: Arrays, Linked Lists, Stacks, Queues, Trees
  • OOP using C++: Classes, Objects, Inheritance, Polymorphism
  • Computer Networks: OSI Model, TCP/IP, Routing
  • Database Management: SQL, ER Diagrams, Normalization
  • Operating Systems: Process Management, Memory Management

Third Year (TE)

  • Software Engineering: SDLC, Agile, UML
  • Web Technologies: HTML, CSS, JavaScript, Node.js
  • Machine Learning: Supervised/Unsupervised Learning, Regression
  • System Programming: Assemblers, Compilers, Linkers
  • Information and Cyber Security: Cryptography, Network Security

Final Year (BE)

  • Artificial Intelligence: Search algorithms, reasoning, and knowledge representation
  • Big Data Analytics: Hadoop, Spark, Data Mining
  • Electives: Various topics like NLP, Cloud Computing, IoT
  • Project Work: Final year capstone project based on real-world problems
  • Seminar: Research and presentation on recent IT topics
